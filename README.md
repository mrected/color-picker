## HSLA Color Picker

Uses React.js, css and javascript to set a color box.

User can use sliders, manualy enter values or click a randomize button to choose colors

<p><strong>Uses:</strong></p>
<ul>
  <li>HTML 5</li>
  <li>CSS</li>
  <li>JavaScript</li>
  <li><a href="http://reactjs.org">React.JS</a></li>
</ul>
